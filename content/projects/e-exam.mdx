A secure full-stack web application for **managing and conducting online examinations**.
It integrates authentication, role-based access control (RBAC), exam management, scheduling, and real-time updates.

## Features
- User authentication & authorization
- Role-based access control (Admin, Teacher, Student)
- Exam scheduling and grading
- Real-time updates (WebSockets)
- Responsive design

## Tech Stack
**Frontend:** Next.js 14, TypeScript, Tailwind CSS  
**Backend:** NestJS, Prisma ORM, PostgreSQL  
**Deployment:** Docker & Docker Compose
